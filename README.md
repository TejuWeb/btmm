# Bauddha Tarun Mitra Mandal (BTMM) Web Portal

A high-performance, aesthetically premium, and modern web application for the Bauddha Tarun Mitra Mandal village community organization.

## 🚀 Technologies Used
- **Next.js 15+**: For high-performance routing and SEO.
- **Tailwind CSS v4**: For the latest modern design system and styling.
- **Framer Motion**: For premium interactive animations and scroll effects.
- **Lucide Icons**: For crisp, modern iconography.
- **TypeScript**: For robust and maintainable code.

## ✨ Key Features
- **Responsive Landing Page**: Seamless experience across mobile, tablet, and desktop.
- **Premium Design**: Glassmorphism, vibrant gradients, and modern typography (Outfit & Inter).
- **Marathi Support**: Fully optimized for Marathi language rendering.
- **Interactive Sections**:
    - **Hero**: Dynamic entrance with organization stats.
    - **About**: interactive view of organization pillars.
    - **Events**: Color-coded upcoming and past event cards.
    - **Members**: Professional executive committee directory.
    - **Gallery**: Modern masonry grid with hover effects.
    - **Contact**: Functional contact form UI and info.
- **Admin Dashboard**: Sleek, modern management interface for organization leadership.

## 🛠️ Getting Started

### Prerequisites
- Node.js (Latest LTS version recommended)
- npm (comes with Node.js)

### Installation
1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📁 Project Structure
- `src/app/`: Next.js App Router (Pages & Layouts)
- `src/components/`: Reusable UI components
- `src/lib/`: Utility functions (e.g., tailwind-merge)
- `src/app/globals.css`: Tailwind v4 design system and custom styles

## 📜 Metadata & SEO
Optimized for social sharing and search engines with custom metadata in `layout.tsx`.
